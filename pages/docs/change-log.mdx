# Changelog

## Version 1.2.0 - March 22, 2024

### New Features

**Reports Page with Export Options** (March 22, 2024)

- Added a comprehensive reports page with flexible date range filtering
- Implemented PDF export functionality using jsPDF for formatted reports
- Added CSV export option for data analysis in spreadsheet applications
- Enhanced date range picker with support for yearly, monthly, and custom date ranges
- Improved documentation with detailed usage guides for the reports feature
- Updated from beta to stable release (1.2.0)

## Version 1.1.1-beta - March 21, 2024

### New Features

**Enhanced Expense Management Page** (March 21, 2024)

- Rebuilt the expenses page with improved filtering capabilities
- Added monthly navigation with visual expense tracking
- Implemented detailed expense summaries with category breakdown
- Enhanced expense table with improved sorting and filtering options
- Added expense actions for quick editing and deletion

## Version 1.1.0-beta - September 28, 2024

### New Features

**Category Groups and Drag-and-Drop** (September 28, 2024)

- Introduced the ability to manage categories using drag-and-drop functionality.
- Added `CategoryGroup` model to the system, allowing users to group categories for better organization.
- Replaced the category table with drag-and-drop interaction for easier management of categories and groups.
- Updated the category icon selection to use an emoji picker for greater flexibility.
- Bumped version to 1.1.0-beta.

**Improved Payment Method and Expense Notes** (September 25, 2024)

- Added new payment methods, including:
  - `CREDIT_CARD`
  - `DEBIT_CARD`
  - `CASH`
  - `BANK_TRANSFER`
  - `DIGITAL_BANK`
  - `SAVINGS`
  - `OTHER`
- Simplified expense notes to allow only one note per expense instead of multiple.
