# Managing Categories

Learn how to create and customize categories to organize your expenses effectively.

<Steps>
### Step 1: Access Category Settings

Navigate to **Settings** in the sidebar, then go to the **Categories** tab. This is where you can manage your expense categories.

### Step 2: Create a New Category

Click on **Add Category**. Enter a name for the category, and select an icon and color to represent it visually.

### Step 3: Edit or Delete Categories

To modify an existing category, click the edit icon next to it. You can rename, change the icon or color, or delete the category if it's no longer needed.

</Steps>
